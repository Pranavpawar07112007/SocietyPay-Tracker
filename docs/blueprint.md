# **App Name**: SocietyPay Tracker

## Core Features:

- Member List: Display a list of society members (10 members).
- Payment Input: Input fields to record payment amount and date paid.
- Local Storage Simulation: Simulate saving user input data to the browser's local storage.
- Payment Status Indicator: Visually highlight members who have paid, to differentiate them from those who haven't. Optionally filter based on the pay status.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), to evoke a feeling of trust.
- Background color: Very light blue (#E3F2FD). A light scheme ensures maximum readability and approachability.
- Accent color: Soft green (#A5D6A7), for positive affirmation (payments received).
- Font: 'PT Sans', a versatile sans-serif, for both headlines and body text.
- Simple, clear icons for payment status (e.g., a checkmark for paid, an exclamation mark for unpaid).
- Clean, tabular layout for member list and payment information.
